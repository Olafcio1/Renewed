## 🔥 Usage
1. Install ATLauncher
2. Log in to your Microsoft account (for playing Minecraft)
3. Create a new instance *("Create Pack" tab)*
   - with the Minecraft version set as 1.3.2
   - with the ModLoader set as Legacy Fabric
4. Open the instance folder *("Instances" tab)*
5. Go to the `mods` subfolder
6. Drag the Renewed [mod jar](https://github.com/Olafcio1/Renewed/actions/) there
